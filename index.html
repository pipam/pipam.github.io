Pipam is package manager... manager

Pipam provides a nice, common UI for using package managers. Want detailed information about packages at your will? Pipam can do that. Like GUIs better than terminals? Pipam is just what you need. Just want a common interface for managing packages? Pipam excels at that 😉.

All of Pipam's core functionality is based on plugins. Pipam plugins provide interfaces to package managers for common features like installing, listing, searching, etc. And it's really easy to write plugins, too!
